Spec Command Attributes

Attribute

Usage

Name

Sets the name used in the workflow. Set to something short or it will truncate (the tooltip shows the full name in this case).

Commands

The individual commands to run, the result of each of which may be read back into a separate macro.

Spec Macro

Reference a spec macro file with this parameter or start a new file. Then right click on the editor and open the editor. Values from up-stream can be substituted in.
